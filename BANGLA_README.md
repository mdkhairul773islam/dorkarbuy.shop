# ই-কমার্স প্ল্যাটফর্ম - বই এবং কোর্স বিক্রয়ের জন্য

## যা তৈরি করা হয়েছে

আমি আপনার জন্য একটি সম্পূর্ণ ই-কমার্স সিস্টেম তৈরি করেছি যেখানে বই এবং কোর্স বিক্রয় করা যাবে।

### ✅ অ্যাডমিন প্যানেল (Filament)

1. **ড্যাশবোর্ড** - সমস্ত পরিসংখ্যান দেখতে পারবেন:
   - মোট আয়
   - আজকের আয়
   - মোট অর্ডার
   - পেন্ডিং অর্ডার
   - সম্পন্ন অর্ডার
   - সক্রিয় পণ্য

2. **ইউজার ম্যানেজমেন্ট** - ব্যবহারকারী পরিচালনা

3. **ক্যাটাগরি ম্যানেজমেন্ট** - পণ্যের ক্যাটাগরি তৈরি ও পরিচালনা

4. **প্রোডাক্ট ম্যানেজমেন্ট** - বই ও কোর্স যোগ করুন:
   - নাম, বর্ণনা, ছবি
   - মূল্য ও ডিসকাউন্ট মূল্য
   - স্টক সংখ্যা
   - লেখক তথ্য
   - সময়কাল/পৃষ্ঠা সংখ্যা
   - ফিচার্ড প্রোডাক্ট মার্ক করা

5. **অর্ডার ম্যানেজমেন্ট** - সকল অর্ডার দেখুন ও পরিচালনা করুন

6. **পেমেন্ট ট্র্যাকিং** - পেমেন্ট স্ট্যাটাস মনিটর করুন

### ✅ কাস্টমার ফ্রন্টএন্ড (Inertia + React)

1. **হোম পেজ** - ফিচার্ড পণ্য এবং ক্যাটাগরি ব্রাউজ করুন

2. **প্রোডাক্ট লিস্টিং** - সকল পণ্য দেখুন:
   - ক্যাটাগরি অনুযায়ী ফিল্টার
   - টাইপ অনুযায়ী ফিল্টার (বই/কোর্স)
   - পেজিনেশন

3. **প্রোডাক্ট ডিটেইল** - পণ্যের বিস্তারিত তথ্য

4. **শপিং কার্ট** - পণ্য যোগ, আপডেট, মুছে ফেলা

5. **চেকআউট** - অর্ডার সম্পন্ন করুন

6. **অর্ডার ট্র্যাকিং** - অর্ডার নম্বর দিয়ে ট্র্যাক করুন

### ✅ পেমেন্ট মেথড

- **bKash** (বিকাশ)
- **Rocket** (রকেট)
- **Nagad** (নগদ)
- **Cash on Delivery** (ক্যাশ অন ডেলিভারি)

## কীভাবে চালাবেন

### ১. ডাটাবেস সেটআপ

```bash
cd /home/khairul/Documents/Ebook/filament-app
php artisan migrate:fresh --seed
```

এটি তৈরি করবে:
- অ্যাডমিন ইউজার: `admin@example.com` / `password`
- টেস্ট ইউজার: `test@example.com` / `password`
- ৪টি ক্যাটাগরি এবং ২০টি পণ্য

### ২. ফ্রন্টএন্ড বিল্ড

```bash
npm run build
```

অথবা ডেভেলপমেন্টের জন্য:
```bash
npm run dev
```

### ৩. অ্যাপ্লিকেশন চালান

```bash
php artisan serve
```

### ৪. ওয়েবসাইট দেখুন

- **ফ্রন্টএন্ড**: http://localhost:8000
- **অ্যাডমিন প্যানেল**: http://localhost:8000/admin
  - ইমেইল: `admin@example.com`
  - পাসওয়ার্ড: `password`

## ডাটাবেস স্ট্রাকচার

তৈরি করা টেবিল:
- `users` - ব্যবহারকারী
- `categories` - ক্যাটাগরি
- `products` - পণ্য (বই ও কোর্স)
- `orders` - অর্ডার
- `order_items` - অর্ডারের আইটেম
- `payments` - পেমেন্ট রেকর্ড
- `carts` - শপিং কার্ট
- `cart_items` - কার্টের আইটেম

```
resources/
├── js/
│   ├── Layouts/        # React লেআউট
│   └── Pages/          # Inertia পেজ
└── views/              # Blade টেমপ্লেট

database/
├── migrations/         # মাইগ্রেশন
└── seeders/           # সিডার
```

## পরবর্তী ধাপ (ঐচ্ছিক)

আপনি চাইলে এই ফিচারগুলো যোগ করতে পারেন:

1. **ইমেইল নোটিফিকেশন** - অর্ডার কনফার্মেশনের জন্য
2. **SMS নোটিফিকেশন** - বাংলাদেশী SMS গেটওয়ে দিয়ে
3. **রিভিউ সিস্টেম** - কাস্টমাররা রিভিউ দিতে পারবে
4. **উইশলিস্ট** - পছন্দের পণ্য সেভ করা
5. **ইনভয়েস জেনারেশন** - PDF ইনভয়েস
6. **রিফান্ড প্রসেসিং** - রিফান্ড ম্যানেজমেন্ট

## গুরুত্বপূর্ণ নোট

⚠️ **প্রোডাকশনে যাওয়ার আগে**:
- ডিফল্ট অ্যাডমিন পাসওয়ার্ড পরিবর্তন করুন
